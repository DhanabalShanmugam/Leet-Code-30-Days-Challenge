# Problem

__Valid Perfect Square__

Given a positive integer num, write a function which returns True if num is a perfect square else False.

Note: Do not use any built-in library function such as sqrt.

Example 1:</br>
Input: 16</br>
Output: true</br>

Example 2:</br>
Input: 14</br>
Output: false</br>
